# Payment-API

## Description

Built using [Nest](https://github.com/nestjs/nest).

```
URL: https://okra-payment.herokuapp.com/
```

Postman collection is in repo, download and import into postman

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
